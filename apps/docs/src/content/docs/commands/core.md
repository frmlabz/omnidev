These commands are always available.

## `omnidev init`

Initialize OmniDev in the current project.

```bash
omnidev init
```

Common usage:

```bash
omnidev init claude-code,cursor
```

## `omnidev sync`

Fetch capability sources and regenerate provider files.

```bash
omnidev sync
```

## `omnidev doctor`

Validate your configuration and project setup.

```bash
omnidev doctor
```
