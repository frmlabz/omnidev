Initialize OmniDev in the current project.

## Usage

```bash
omnidev init
```

## With providers

Initialize with specific providers enabled:

```bash
omnidev init claude-code,cursor
```

## What it does

- Creates an `omni.toml` configuration file
- Creates the `.omni/` directory structure
- Sets up default profile configuration
